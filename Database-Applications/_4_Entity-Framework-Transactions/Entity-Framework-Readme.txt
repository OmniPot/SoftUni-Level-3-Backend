In order to run all the programs you need to:
1. Right click on the solution while in Visual Studio
2. Click on "Enable NuGet Package Restore"
3. Click "Yes" in the prompt
4. CLick "Yes"
5. Right click on the solution and "Rebuild Solution"
6. Change every Solution project's App.config file's connection string propery to match your SQL server settings.

Good Luck :)
